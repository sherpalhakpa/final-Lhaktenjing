# Films application with RESTful Backend and Frontend

## Backend

- `GET		/api/films`
- `GET		/api/films/:id`
- `GET		/api/categories`
- `GET		/api/categories/:id`
- `POST		/api/films`
- `POST		/api/categories`
- `PUT		/api/films`
- `DELETE	/api/films`
- `GET		/api/films-categories`

All resources return JSON.

## Frontend

- `index.html`
- `films.html`
- `show-film.html`
- `insert-film.html`
- `update-film.html`
- `delete-film.html`
- `categories.html`
- `show-category.html`
- `insert-category.html`

Frontend uses Bootstrap
